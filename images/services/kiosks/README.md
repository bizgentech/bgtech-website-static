# Self-Service Kiosk Solutions Images

This folder contains images for the Self-Service Kiosk Solutions page and related content.

## Image Guidelines

### Hero/Banner Images
- **Recommended Size:** 1920x1080px or 1600x900px
- **Format:** JPG (optimized) or WebP
- **Max File Size:** 500KB
- **Content:** Modern self-service kiosks in retail, banking, hotel check-in, restaurant ordering

### Kiosk Hardware Images
- **Recommended Size:** 800x1200px (portrait) or 1200x800px (landscape)
- **Format:** JPG or WebP
- **Max File Size:** 300KB
- **Content:** Touchscreen kiosks, payment terminals, check-in stations, queue management systems

### Industry-Specific Images
- **Recommended Size:** 1200x800px
- **Format:** JPG or WebP
- **Content:** Retail kiosks, banking ATMs, hotel check-in, restaurant self-order, wayfinding

### Interface Screenshots
- **Recommended Size:** 1080x1920px (kiosk portrait) or 1920x1080px (landscape)
- **Format:** PNG or JPG
- **Content:** Kiosk UI designs, touchscreen interfaces, payment flows

## Naming Convention

**Required File:**
- `hero.jpg` - Main hero image for the service page (1920x1080px, 300-500KB)

**Additional Images:**
Use descriptive, lowercase names with hyphens:
- `banking-self-service.jpg`
- `hotel-check-in-kiosk.jpg`
- `restaurant-ordering.jpg`
- `payment-terminal.jpg`
- `touchscreen-interface.png`
- `feature-{name}.jpg` - For feature sections

## Current Usage

Images from this folder are used on:
- `/services/kiosks` - Service detail page
  - `hero.jpg` - **REQUIRED** - Main hero background image
  - Industry-specific kiosk images (optional)

**Note:** Home page carousel uses separate images from `/images/hero/`
