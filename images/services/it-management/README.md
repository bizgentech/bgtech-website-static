# IT Management Services Images

This folder contains images for the IT Management Services page and related content.

## Image Guidelines

### Hero/Banner Images
- **Recommended Size:** 1920x1080px or 1600x900px
- **Format:** JPG (optimized) or WebP
- **Max File Size:** 500KB (compress for web)
- **Content:** IT professionals, servers, network operations centers, support teams, monitoring dashboards

### Feature/Section Images
- **Recommended Size:** 800x600px or 1200x800px
- **Format:** JPG or WebP
- **Max File Size:** 300KB
- **Content:** Microsoft 365 administration, cybersecurity, help desk, network management

### Icons/Graphics
- **Recommended Size:** 512x512px or 1024x1024px
- **Format:** PNG (with transparency) or SVG
- **Content:** Service icons, technology logos, feature illustrations

## Naming Convention

**Required File:**
- `hero.jpg` - Main hero image for the service page (1920x1080px, 300-500KB)

**Additional Images:**
Use descriptive, lowercase names with hyphens:
- `microsoft-365-admin.jpg`
- `cybersecurity-monitoring.jpg`
- `help-desk-support.jpg`
- `network-infrastructure.jpg`
- `feature-{name}.jpg` - For feature sections

## Current Usage

Images from this folder are used on:
- `/services/it-management` - Service detail page
  - `hero.jpg` - **REQUIRED** - Main hero background image
  - Additional images for content sections (optional)

**Note:** Home page carousel uses separate images from `/images/hero/`
