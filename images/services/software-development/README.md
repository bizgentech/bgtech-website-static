# Custom Software Development Images

This folder contains images for the Custom Software Development service page and related content.

## Image Guidelines

### Hero/Banner Images
- **Recommended Size:** 1920x1080px or 1600x900px
- **Format:** JPG (optimized) or WebP
- **Max File Size:** 500KB
- **Content:** Developers coding, modern office spaces, React/Next.js code on screens, collaborative development

### Feature/Section Images
- **Recommended Size:** 800x600px or 1200x800px
- **Format:** JPG or WebP
- **Max File Size:** 300KB
- **Content:** Web applications, mobile apps, code editors, API integrations, modern UI/UX designs

### Screenshots
- **Recommended Size:** 1440x900px (desktop) or 375x812px (mobile)
- **Format:** PNG or JPG
- **Content:** Application interfaces, dashboards, user flows

### Technology Stack Images
- **Recommended Size:** 512x512px
- **Format:** PNG (with transparency) or SVG
- **Content:** React, Next.js, NestJS, PostgreSQL logos/icons

## Naming Convention

**Required File:**
- `hero.jpg` - Main hero image for the service page (1920x1080px, 300-500KB)

**Additional Images:**
Use descriptive, lowercase names with hyphens:
- `web-app-dashboard.jpg`
- `mobile-app-interface.png`
- `react-nextjs-code.jpg`
- `api-integration.jpg`
- `feature-{name}.jpg` - For feature sections

## Current Usage

Images from this folder are used on:
- `/services/custom-software` - Service detail page
  - `hero.jpg` - **REQUIRED** - Main hero background image
  - Additional screenshots and feature images (optional)

**Note:** Home page carousel uses separate images from `/images/hero/`
