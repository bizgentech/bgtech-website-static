# Industries We Serve Images

This folder contains images and graphics representing the various industries BG Tech serves.

## Image Guidelines

### Industry Hero Images
- **Recommended Size:** 1200x800px
- **Format:** JPG or WebP
- **Max File Size:** 300KB
- **Content:** Industry-specific scenes, professional environments

### Industry Icons/Graphics
- **Recommended Size:** 512x512px or 1024x1024px
- **Format:** PNG (with transparency) or SVG
- **Max File Size:** 100KB
- **Content:** Industry-specific icons, illustrations

## Industries Covered

1. **Legal Firms**
   - Law offices, legal professionals, courtrooms
   - `legal-firms.jpg`, `legal-icon.svg`

2. **Construction & Architecture**
   - Construction sites, architects, blueprints, testing labs
   - `construction.jpg`, `architecture-icon.svg`

3. **Insurance Companies**
   - Insurance offices, professionals, claims processing
   - `insurance.jpg`, `insurance-icon.svg`

4. **Car Dealerships**
   - Showrooms, automotive sales, service centers
   - `automotive.jpg`, `dealership-icon.svg`

5. **Retail & Banking**
   - Retail stores, bank branches, customer service
   - `retail-banking.jpg`, `finance-icon.svg`

6. **Small-Medium Businesses**
   - Generic office environments, SMB operations
   - `smb.jpg`, `business-icon.svg`

## Naming Convention

Use industry name with type:
- `{industry-name}-hero.jpg` - For large images
- `{industry-name}-icon.svg` - For icons/graphics
- `{industry-name}-feature.jpg` - For feature highlights

## Current Usage

Images from this folder are used on:
- Home page "Industries We Serve" section
- About page industry showcase
- Individual industry landing pages (if created)

**Note:** Currently using React Icons (HiScale, HiOfficeBuilding, etc.).
Custom images/icons can enhance brand identity.
