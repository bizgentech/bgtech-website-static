# CondoLink Platform Project Images

This folder contains images for the CondoLink Platform portfolio case study.

## Project Overview

Complete condominium management solution with owner/tenant portals, automated rent collection, and maintenance tracking.

## Image Guidelines

### Hero/Banner Image
- **Recommended Size:** 1920x1080px or 1600x900px
- **Format:** JPG or WebP
- **Max File Size:** 500KB
- **Content:** Modern condominium building, property management office, residential complex

### Application Screenshots
- **Recommended Size:** 1440x900px (desktop) or 375x812px (mobile)
- **Format:** PNG (recommended for UI screenshots)
- **Max File Size:** 500KB per image
- **Content:**
  - Owner/Tenant portal dashboard
  - Payment processing interface
  - Maintenance request system
  - Document management
  - Communication center
  - Mobile app views

### Feature Highlights
- **Recommended Size:** 1200x800px
- **Format:** PNG or JPG
- **Content:** Rent collection automation, maintenance tracking, resident portal

## Naming Convention

Use descriptive names with project prefix:
- `condolink-hero.jpg`
- `condolink-dashboard.png`
- `condolink-payments.png`
- `condolink-maintenance.png`
- `condolink-portal.png`
- `condolink-mobile.png`
