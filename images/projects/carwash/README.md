# CarWash App Project Images

This folder contains images for the CarWash App portfolio case study.

## Project Overview

On-demand mobile car wash platform connecting customers with service providers through GPS-based matching.

## Image Guidelines

### Hero/Banner Image
- **Recommended Size:** 1920x1080px or 1600x900px
- **Format:** JPG or WebP
- **Max File Size:** 500KB
- **Content:** Mobile car wash service, professional detailing, on-demand service

### Application Screenshots
- **Recommended Size:** 375x812px (mobile) or 1440x900px (web dashboard)
- **Format:** PNG (recommended for UI screenshots)
- **Max File Size:** 500KB per image
- **Content:**
  - Customer mobile app interface
  - Service provider app
  - GPS location matching
  - Booking flow
  - Payment integration
  - Service history
  - Rating system

### Feature Highlights
- **Recommended Size:** 1200x800px
- **Format:** PNG or JPG
- **Content:** GPS matching, real-time tracking, Stripe Connect payments

## Naming Convention

Use descriptive names with project prefix:
- `carwash-hero.jpg`
- `carwash-customer-app.png`
- `carwash-provider-app.png`
- `carwash-booking.png`
- `carwash-gps-match.png`
- `carwash-payment.png`
- `carwash-tracking.png`
