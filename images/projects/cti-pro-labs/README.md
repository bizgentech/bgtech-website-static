# CTI Pro Labs Project Images

This folder contains images for the CTI Pro Labs portfolio case study.

## Project Overview

Laboratory management system for construction testing operations with automated workflows and real-time tracking.

## Image Guidelines

### Hero/Banner Image
- **Recommended Size:** 1920x1080px or 1600x900px
- **Format:** JPG or WebP
- **Max File Size:** 500KB
- **Content:** Construction testing lab, laboratory equipment, testing operations

### Application Screenshots
- **Recommended Size:** 1440x900px (desktop) or 375x812px (mobile)
- **Format:** PNG (recommended for UI screenshots)
- **Max File Size:** 500KB per image
- **Content:**
  - Dashboard overview
  - Sample tracking interface
  - Test report generation
  - Mobile PWA interface
  - Workflow automation screens

### Feature Highlights
- **Recommended Size:** 1200x800px
- **Format:** PNG or JPG
- **Content:** Key features, UI components, data visualizations

## Naming Convention

Use descriptive names with project prefix:
- `cti-hero.jpg`
- `cti-dashboard.png`
- `cti-sample-tracking.png`
- `cti-mobile-app.png`
- `cti-reports.png`
- `cti-workflow.png`

## Current Usage

Images from this folder are used on:
- `/portfolio/cti-pro-labs` - Full case study page
- Home page Featured Projects section (card placeholder)
- Portfolio overview

## Tech Stack

React, NestJS, PostgreSQL, PWA
